API Zip Codes by Radius

Use this API to find all zip codes within a given radius of a zip code. Send a GET request to https://zipcodedistanceapi.redline13.com/rest/<api_key>/radius.<format>/<zip_code>/<distance>/<units>?minimal.

If you only want the zip code in the response, you can add a GET parameter of minimal to the url. For example, https://zipcodedistanceapi.redline13.com/rest/<api_key>/radius.csv/08057/5/miles?minimal.
